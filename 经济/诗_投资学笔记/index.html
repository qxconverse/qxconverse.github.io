投资和投资学

  1. 炒股（股市牛转熊后，炒房了，房价上涨
  2. 炒房
  3. 买债券、基金、期货、期权
  4. 投资建厂

投资的特性

  1. 延迟消费
  2. 期望获得收益
  3. 不确定性和风险

投资的定义：为了可能不确定的将来的消费（价值）而牺牲现在一定的消费（价值）

投资学能学到什么

  1. 市场和工具介绍
  2. 资产定价
  3. 资产组合管理

不是什么

  1. 工具书
  2. 培训课

投资与投资主体

投资主体

  1. 家庭（家庭收入=投资（直接投资、间接投资）+消费（购买商品或服务））
  2. 企业（收入=投资（研发、培训、建造厂房、购买设备、存货投资）+消费（发工资）+融资（如发股票））
  3. 政府（税收+融资=投资（直接、间接投资）+支出（医疗等社会保障、公务员支出））

关系

直接投资和间接投资的区别

资产负债表

金融泡沫

股票等金融资产价格上涨会导致

  1. 家庭将更多的收入配置到金融资产
  2. 企业融资或变卖实物资产投入到金融资产
  3. 银行的货币创造进一步推升金融资产价格

实物资产与金融资产要平衡很重要

投资与虚拟经济

  1. 家庭将资金配置金融资产
  2. 企业将融资等所得资金配置金融资产
  3. 资金闭环但没有创造任何商品或服务

虚拟经济的适度发展对实体经济的发展也有推动作用

实体经济与虚拟经济的平衡

2.投资与短期经济增长

投资-储蓄恒等式

C（居民消费）+S（居民储蓄）=C（企业所生产产品）+I（企业投资）

加入政府和国际经济体时：C+S+T+Kr=C+I

政府投资案例

现象

  1. M2（货币与准货币）增速飙升
  2. 社会融资规模高速增长
  3. 地方政府债务飙升，杠杆率很高
  4. 企业杠杆率飙升
  5. 居民杠杆率飙升（债务占GDP比重 ）

导致的问题：

  1. 各部门债务高企，违约率大大提高
  2. 产能过剩，急需供给侧改革（消费–，生产++，消费挤压）
  3. 民营企业倒闭，工人失业
  4. 房价飙升（M2增速很快，08、15年飙升），催生一系列民生问题
