投资与经济运行

1、投资和投资学

  1. 炒股（股市牛转熊后，炒房了，房价上涨）
  2. 炒房
  3. 买债券、基金、期货、期权
  4. 投资建厂

投资的特性

  1. 延迟消费
  2. 期望获得收益
  3. 不确定性和风险

投资的定义：为了可能不确定的将来的消费（价值）而牺牲现在一定的消费（价值）

投资学能学到什么

  1. 市场和工具介绍
  2. 资产定价
  3. 资产组合管理

不是什么

  1. 工具书
  2. 培训课

投资主体

  1. 家庭（家庭收入=投资（直接投资、间接投资）+消费（购买商品或服务））
  2. 企业（收入=投资（研发、培训、建造厂房、购买设备、存货投资）+消费（发工资）+融资（如发股票））
  3. 政府（税收+融资=投资（直接、间接投资）+支出（医疗等社会保障、公务员支出））

关系

直接投资和间接投资的区别

资产负债表

金融泡沫

股票等金融资产价格上涨会导致

  1. 家庭将更多的收入配置到金融资产
  2. 企业融资或变卖实物资产投入到金融资产
  3. 银行的货币创造进一步推升金融资产价格

实物资产与金融资产要平衡很重要

投资与虚拟经济

  1. 家庭将资金配置金融资产
  2. 企业将融资等所得资金配置金融资产
  3. 资金闭环但没有创造任何商品或服务

虚拟经济的适度发展对实体经济的发展也有推动作用

实体经济与虚拟经济的平衡

托宾Q系数：Q=公司的市场价值/公司的重置成本

2、投资与短期经济增长

投资-储蓄恒等式

C（居民消费）+S（居民储蓄）=C（企业所生产产品）+I（企业投资）

加入政府和国际经济体时：C+S+T+Kr=C+I

储蓄的来源

居民储蓄

企业储蓄

政府储蓄

政府投资案例

现象

  1. M2（货币与准货币）增速飙升
  2. 社会融资规模高速增长
  3. 地方政府债务飙升，杠杆率很高
  4. 企业杠杆率飙升
  5. 居民杠杆率飙升（债务占GDP比重 ）

导致的问题：

  1. 各部门债务高企，违约率大大提高
  2. 产能过剩，急需供给侧改革（消费–，生产++，消费挤压）
  3. 民营企业倒闭，工人失业
  4. 房价飙升（M2增速很快，08、15年飙升），催生一系列民生问题

3、投资与长期经济增长

索洛模型

生产和消费，资本K、劳动L、知识A是生产三大要素。技术进步表现为劳动效率的提高，是最终影响经济增长的因素。

内生增长模型

纳入技术进步后，资本的边际报酬不再呈现递减的特性

案例：供给侧改革

  1. 削减落后产能：影响资本的边际报酬
  2. 体制改革：人口政策和土地政策（影响人口增长率）、行政制度和政府干预改革（影响全要素生产率）
  3. 创新驱动战略：影响技术进步（唯一可持续影响经济发展）

4、投资与经济波动

凯恩斯-塞缪尔森经济周期模型（由消费和储蓄的变动引起）

a的有效需求不足理论：边际消费倾向的作用造成消费需求不足，也造成投资需求的不足，即社会总需求的不足，必然导致社会总就业量的不足和经济危机的产生

b的加速模型：投资的乘数效应、产出的加速效应

实际经济周期模型（由外部因素引起）

熊彼特的创新周期论

投资规模、结构与效率

1、投资规模概述

年度投资规模（短期、流量）

在建投资规模（长期、存量）

2、投资结构

投资主体结构：企业和政府的投资目标、运行机制、对经济影响不一样

企业投资和政府投资的关系：辩证统一（此消彼长、相辅相成）

投资产业结构

主要是第二产业、第三产业

投资的融资结构

预算内资金+国内贷款+利用外资+自筹资金（比例最高）

3、投资效率衡量

资本产出比

资本产出比（cor）、投资产出比（icor）

2个指标，越小说明投资效率越高

资本边际收益率均一化准则

市场有效且达到均衡时，所有部门的资本边际收益率呈现相同的特征

资本的流向和效率

资本配置有效时，资本会从边际收益低的部门向高的部门转移

当经济体可以“在相对高成长的行业中增加投资，而在衰退的部门中减少投资”，就认为该经济体的投资是有效率的
