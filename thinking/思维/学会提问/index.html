学会如何提问

发表于 2021-05-03 | 更新于: 2021-05-03 | 分类于 思维 | 浏览 次
字数统计: 638 | 阅读时长 ≈ 2

发现问题比解决问题更重要，且更难；我们无法在制造问题的同一个思维层次上解决这个问题

与人沟通问题时，多用问号，少用句号或肯定的语句

提问的几个价值

  1. 帮助我们找到问题背后的本质问题
  2. 帮助我们整理杂乱信息，提升工作生活效率
  3. 好的提问能激发创新
  4. 改善人际关系
  5. 让自己和他人变得更积极，而非消沉

问题的定义（3者叠加，问题就出现了）

  1. 现状和期望相差较大
  2. 不知道原因
  3. 急需处理

提问6步法

  1. 发现问题（升维法、逆向思维法，去发现问题）
  2. 根问题界定（问题空间）
  3. 根因确认（询问目标和现状的具体差距是什么，大分小；树状分析法，对原因进行重排，形成树状，再找根节点）
  4. 策略制定(围绕问题根因，从多个维度提出解决方案；对可能的策略进行有效提问，可以使用5W1Htiwe恩发)
  5. 策略评估（优先排序矩阵法、n宫格/象限法，如从重要性、可操作性2个维度）
  6. 落实行动（步骤、责任人、支持、完成时间、意愿度）
问题空间
  1. 需要解决的问题是什么、问题现状
  2. 你为什么需要解决这个问题
  3. 如果问题解决，理想结果是什么
  4. 问题的子问题，问题的构成
  5. 曾经尝试哪些解决方法、效果如何
  6. 问题解决需要哪些约束条件
3个职场提问锦囊
  1. 开拓思维，站在更高的维度去提问
  2. 利用提问，引发对方深刻思考
  3. 在之前的提问基础上不断追问
几种比较有力的提问方式
  1. 是什么、定义（如何定义客户、这是什么客户）
  2. 对比（A和B的区别是什么）
  3. 解释性（发生了什么）
  4. 因果中的因（为什么产品系列少会导致业绩下降）
  5. 因果中的果（为什么晚睡会导致第二天没有精神）
  6. 程序性（如何完成产品研发流程的）
  7. 推理性（什么会导致用户流失）
  8. 预测性（如果2个部门合并，会发生什么）

我们无法在制造问题的同一个思维层次上解决这个问题

NLP逻辑思维层次（数字越小，层次越高）
  1. 精神、系统
  2. 身份
  3. 信念、价值观
  4. 能力
  5. 行为、行动
  6. 环境
