SLI指的是服务质量指标

常见的有：请求延迟（处理请求所消耗的时间）、错误率（请求处理失败的百分比）、系统吞吐量（每秒请求数）。还有一个可用性，代表服务可用时间的百分比。

标准化日志示例

一些日志记录规则：

  • 包括时间戳
  • 格式为 JSON
  • 请勿记录无意义的事件
  • 请记录所有应用程序错误
  • 可以有日志警告
  • 打开日志记录
  • 以可读的形式写消息
  • 请勿在生产中记录信息数据
  • 请勿记录无法阅读或反应的任何内容
