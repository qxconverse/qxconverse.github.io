Java探针SpringBoot项目

通过本博客，您能完成的目标是

  1. 创建一个javaagent项目，并打包成探针包（agent包）
  2. 对已有的SpringBoot项目中的某个方法计算其耗时

需要的环境

  1. Idea开发环境
  2. javassist依赖包
  3. SpringBoot web项目

核心为MyAgent类和MyTransformer类，第一个类可以理解为agent的入口，需要定义一个premain的方法，方法里面加入一个自定义的transformer类，可以理解为写入代码的具体逻辑。

具体的代码操作，这里是在某个方法执行之后打印出该方法执行的时间。当找到该类后，再获取某个方法，接着写入加入的代码，就像我们正常写的代码那样，区别在于，传入的是代码字符串，比如beginTime = System.currentTimeMillis();。

附：遇到的坑

在进行javaagent的实践时，一开始mvn package打出的包，在执行命令后，发现代码执行到javassist之前就没有下文了，最后意识到，可能是因为jar包中没有javassist的依赖。这个问题隐藏很深，因为没有任何报错。后来依据这篇博文https://www.jianshu.com/p/9c284ed187e0，将maven依赖打入jar包。

后来遇到了无法加载类的问题，参考这篇博客解决了：https://www.kagura.me/dev/20190923102501.html

如果遇到"Invalid constant type 18”的问题，则替换mvn依赖版本，javassist要改成3.20版本

最后遇到的问题javassist Type top (current frame, locals[2]) is not assignable to long，参考了这篇博客解决了：https://stackoverflow.com/questions/34223471/how-to-get-arround-java-lang-verrifyerror，原来是加入的代码问题
