最近的一些感想

开始写作

其实只是想写一些之前就已经悟出的一些道理。这些道理应当足够地简单明了。

看清事情的本质

这一点比较笼统了，但是细则都很明了。做到它很难，特别是研究一些科学理论时，但是在生活中，会变得简单很多。

还记得很久之前在知乎上看到的问题：你觉得打篮球的男生酷么？很多回答会从男生打篮球这一行为来分析并回答。但当我看到了一个简短的回答，觉得很符合现实。帅气的男生打篮球叫酷，丑的男生打篮球啥都不是。这个视角我觉得是看清了事情的本质，你会发现重要的不是打篮球，而是帅不帅的问题，帅气的男生干什么都可以用一大堆褒义词形容。还有其它类似的例子，比如男生爱讲黄段子好么？同样的逻辑，帅气的男生讲黄段子叫幽默风趣；丑的男生讲黄段子那叫猥琐。讲黄段子也是要看主体是谁才能分析。

还有很多很多这样的例子。

再拿最近的很多直播带货的例子来说。为什么像罗永浩、董明珠等人直播带货能有那么大的成交量。可能你会觉得，是因为他们擅长于销售与推广。这里很容易就能抓住漏洞，比他们擅长销售和推广的人大有人在，但是他们并不能带来巨大的成交量。随便拉一个淘宝的卖家来直播，在带货方面的技巧都要高于他们。其实原因就在于，是带货人的问题，找影响力大的、舆论度高的人来带货自然能够吸引更多的人。

同样的，生活中，地位高的人（职级高或名望大或由其它方面带来优势的人）不管说什么都是对的；不在于他说了什么，而在于他是谁。

滚雪球效应（优势效应）

这点从字面意义上就明了，雪球会越滚越大，且越大的雪球，滚出来的增量也越大。

表面效应（肤浅效应）

人是感官动物，人有意识、有主观能动性，有思维。但人也是天性懒惰的。

所以，有人问：身材、穿衣打扮和能力哪个重要。

身材会更重要些，如果身材不足，则后面2个来凑。因为身材基本上是与生俱来的。且再加上滚雪球效应，身材好的人自带自信，再来发展其它方面也会更快些；毕竟，就像手里有了老本，干什么事都不会有任何顾虑，这样反而能干好其它很多事情。否则，相当于0装备起身，建立自信的过程、发展其它方面的能力也会艰难些。

让步效应

这里想到了点外卖的很多事情。经过多年，我们发现外卖的折扣越来越多了，但是价格却越来越贵。商家在这里用到的就是这样的效应。

举个例子，一份外卖的制作成本可能只有8元。在过去，优惠较少的时候，可能会有满15元，减3元的优惠。把定价设置为16元。最终你需要花13元吃一顿外卖。心里还说得过去。

而现在，一份外卖的制作成本可能涨到了10元。但是各种优惠满天飞，比如满30减15元，满50减20元等。还有打5折优惠。但是你会发现，售价变得离谱起来，38元的售价。这时你用了券后，减去15元后，23元，再加上餐盒费、派送费，吃一顿饭也得接近30元。于是，就去找打5折的，却发现5折菜品的原价都是40几元，你又被告知，优惠券和折扣不能同时享用。所以，你为了点外卖而进行的各种计算，商家早就预习好了。你吃完发现，这和原来的16元外卖是一样的啊！

这些变来变去的东西很难掌握，但其中有不变的道理。就是，不管怎么，商家一定要赚到一定的钱才会去经营。如今的物价早已高出之前很多。如果还是按照之前的卖法，那怕是要倒闭掉。于是才会想尽各种办法，加上很多其它策略把那些钱挣回来。同时，你会发现，虽然外卖的价格变高了，但其实我们的薪资较几年前也有涨幅啊。这样一想，其实会平衡很多。

登门槛效应（又称得寸进尺效应、台阶效应）

这个就是字面意思，得寸进尺，很好理解。

幸存者偏差效应

我们在淘宝等购物网站上，看到的都是身材特别好、长得特别漂亮（帅气）的人试穿的东西。他们穿什么都特别合适。但仅仅是少数的人，对于大多数买家而言，这样的东西可能并不会适合自己。
