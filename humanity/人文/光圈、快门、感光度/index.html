光圈、快门、感光度

发表于 2018-02-14 | 更新于: 2021-07-03 | 分类于 人文 | 浏览 次
字数统计: 827 | 阅读时长 ≈ 2

在理解的光圈和快门的时候，拿眼睛做对比学习效果较好

光圈

  1. 镜头就是你的眼睛，眼睛可以睁大、睁小。人们说的大眼睛就好比大光圈；而说的眯眯眼，就好比小光圈。

  2. 在相机中，通常显示数值，数值越小，光圈越大。比如4.5就是大光圈，而18、22、32等属于小光圈。

  3. 当光线充足时，比如面对太阳光。我们通常会眯眼睛，所以此时适合使用小光圈，减少曝光量。

  4. 从景深的角度分析：当我们想看得很远时，我们会眯眼睛。或者把一本书从远处慢慢往眼前移动，我们要慢慢眯眼才能一直看清楚。假设书摆在你很靠近眼睛的地方。你的眼睛会很不舒服，就会想眯眼或是直接闭眼。所以小光圈往往使得景深较深，即不管远处还是近处都比较清晰。而光圈越大，就会更加凸显近景，而模糊远景。

快门

  1. 这个时候我们拿眨眼睛做对比。快门控制曝光的时间。快门越快，代表着很快地眨眼睛。而快门越慢，好比很慢地眨眼睛。

  2. 和光圈相反，在相机中，快门数值越大说明，快门越快。否则越慢。比如3200，即快门时间为1/3200秒，这已经是很快的速度了。在拍摄时，如果想抓住瞬间，比如高速移动的物体，那么要提高快门。

感光度

先谈一谈曝光

影响曝光的3个因素为光圈、快门、感光度。

光圈控制镜头进光直径的大小，光圈越大曝光越多。快门控制通光时间的长短，快门时间越长曝光越多。

感光度这个对比眼睛很难接受。简单来说：感光度越高，越敏感。也就是说同等亮度的环境下，感光度设置越高，需要的曝光就越少。

拍摄夜景

  1. 小光圈

  2. 三脚架

  3. 较低的iso，即感光度

  4. 光圈优先，长时间曝光，即慢速快门

光圈优先、快门优先

光圈越小，快门越慢（光圈数值越大，快门数值越小）其实可以这样理解：假设光量一定，光圈越小，就要更多的时间去获取光，所以快门越慢。

如何提高摄影技术

  1. 多去欣赏专业摄影师的照片，并去模仿

  2. 仔细研究优秀照片的优秀之处，是因为光线，模特，还是构图…….

  3. 只展示自己最好的照片，质量大于数量，要有耐心与意志力

  4. 侧重风光摄影

  5. 尽可能地多学习自己所喜欢的摄影类型和风格

  6. 行动起来

构图的7个技巧

  1. 三分法则，这个方法是将照片在水平和竖直方向分别平分为3份，这9条平分线和它们的交点就是关键的位置

  2. 风景构图，要有层次感，增加纵深

  3. 运用线条

  4. 对比与反差，比如黑与白

  5. 简洁，突出重点

  6. 水平线和垂直线能给照片增加强烈的结构感

  7. 形状
