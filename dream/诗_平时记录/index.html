平时记录或总结

Rule

  1. 不懂不害怕，千万不要不懂装懂，知道20%就不要装作知道100%
  2. 别人永远只在乎反对的声音，赞同的声音没有任何意义，只会贬低自我
  3. 对于涉及到的东西，一定要做总结，影响出去，不要一个人知道，尽全力地影响别人
  4. 目前阶段，不要在意好与不好；把想的东西表达出来就可以
  5. 做事；总结、反思；形成自己想法；表达出来；推动其他方

删除方法论

看看自己以后遇到该问题，有没有解决方案。如果有，就删除。如果没有，看是否能分享，是否总结好了，如果已经总结好了，就分享出去。否则，再看有没有用到过或将来需要用，如果没有，就删除；如果有，那就立即总结并分享出去。

整理谷歌收藏夹里面的链接，删除掉没有用的，如何判断是否有用，利用率，哪些是收藏了后再也没有打开过了。哪些是以后基本上用不到的。要有舍才有得，不要什么都留着。转化率的问题。不要看你有什么，而是看你能输出什么。这是一个信息量巨大的时代，你所遇到的大多数问题，别人都遇到过来。所以收藏知识的用处不大，重要的是知道如何去搜索。这是一个有关思维与能力的问题

减少人生无必要的开销，删除多余的东西

消费后的东西一定要充分利用，达到最高的输出率；比如买回的书没有读完，就不要买新书

收藏的必要条件是已经行动过，禁止单纯地收藏

解决问题的能力*职场心态+视野

  1. 确定自己的精神（愿景）

  2. 确定自己的身份（使命）

  3. 确定自己的BVR（信念、价值观、规条）

  4. 确定自己的能力（执行力、视角、思维逻辑、表达能力、硬实力）

  5. 确定自己的行为

执行力的问题有几个原则

  1. 回家买票原则
  2. 先干5分钟原则
  3. 看电视原则
  4. 考试交卷原则

生活的规划

  1. 工作（每天列出todo进行）
  2. 家人（每周至少一次的视频通话）
  3. 住的地方（每周3、4买东西，可以周6、7到；每周末洗一次被子和床单；每天早上一碗粥+1个鸡蛋+1杯牛奶；）
  4. 朋友

想象一下，自己要成为的自己的样子

  1. 日记的去隐私化，每周的生活规划、简单生活、断舍离
  2. 现在不开心的来源
  3. 复盘5年前的生活，与现在的变化
  4. 人生的重点事情、个人okr
  5. 生活家，记录哪些是居家用品、哪些是出门在外的物品；电子产品清单、书籍清单

工作rule

  1. 每天有意识，做得好的、做的不好的
  2. 平时多搞线索（论据），如wiki
  3. 很多问题不要局限在技术层面，要跳到业务上、制度上（人）
  4. 每一件事情，都要知道为什么要做，意义在哪里，强制和业务、人挂钩，技术只是工具！解决方案=人+业务+技术
  5. 平时多关注数据统计，所有的东西都需要有数据的支撑！
  6. 述职核心就是讲自己的故事，一定要突出自己做了什么，不管高端与否，是否完整。
  7. 不要给自己设界限。自己做的事情，不一定是rd的事情，不一定只做业务相关的。
